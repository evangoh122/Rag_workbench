## Featured Project: RAG Workbench
The RAG Workbench is an auditable agent-driven financial analyzer designed to inspect and cross-verify SEC 10-K filings.
* **Launch URL**: `/rag`
* **Core Capabilities**:
  * **Graph RAG**: Visualizes entity relation graphs linking corporate financial facts.
  * **XBRL Verification**: Performs deterministic math verification on raw SEC filing data.
  * **Pipeline Traceability**: Provides a live execution path trace showing every step of the agent's calculations and reasoning.
