# Financial QA Skill

The Financial QA Skill allows agents to query complex corporate disclosures and receive auditable, mathematically verified answers.

## Capabilities
- **SEC Filing Queries**: Search through annual (10-K) and quarterly (10-Q) reports.
- **XBRL Reconciliation**: Cross-references numbers mentioned in answers against database records of filed XBRL facts.
- **Graph RAG Navigation**: Explores relationships between companies, figures, and metrics.
- **Pipeline Auditing**: Inspect the precise steps (retrieval, extraction, verification) of the agent's work.
